SenLaw (Entrepreneurship Project)
April - May 2022

Our project objective for SenLaw was to create a mobile applications that acts as an online portal to connect senior citizens in need of legal help to law firms or lawyers. 
My role in this project was a programmer and tester. I was responsible for developing the back-end API portion of our applications and throughout testing it so the frontend team could easily integrate it.
I had to overcome several challenges for this project, but the main one that I struggled the most with is collaboration and proper testing. Learning to efficiently use GitHub as well as CI (continuous integration) took me a lot of time to learn and master. I also was faced with the problem of testing new API endpoints in a matter of minutes when I was live programming with the frontend team. I ended up abandoning using a JavaScript testing library such as Mocha/Chai or Jest and learned Insomnia to make tests at a fast pace.
After completing this project, I realized how much teamwork and leadership was involved in this project to ensure that we could rapidly program and create a prototype. We had to be clear in our planning and have the same vision so we did not have any conflicting ideas when programming separately.

Student Health Art Project
October 2021

Our projective objective for our Student Health art project was to create a product using Python's Turtle package that had the potential to motivate a student if they were feeling down due to the pressure of the pandemic on their mental and social health. 
My role in this project was a programmer and documentation writer, but roles were not very present in this project as we worked in a team of two and had a lot of overlap. 
Our team, like the aforementioned project, had a lot of challenges regarding communication and collaboration. We initially struggled to communicate effectively and clearly with one another as we had not created any communication channels with each other. Eventually, however, we established a channel and were able to communicate and therefore effectively collaborate outside of class to finish our project on time.
